METERIOT GUIDE

To use METERIOT,Please add following codes in your <head> tag:

<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
<meta http-equiv="x-ua-compatible" content="ie=edge">

This means that: tell the mobile phone to give the actual width of the phone, and tell the Internet Explorer to render as EDGE MODE

Also, to include METERIOT, add following code to <head> tag

<script src="https://cdn.bootcss.com/jquery/3.1.1/core.js"></script>
<script src="https://cdn.bootcss.com/jquery/3.1.1/jquery.min.js"></script>
<script src="js/METERIOTController.core.js"></script>
<link type="text/css" href="css/METERIOT.main.css" rel="stylesheet" />
<link type="text/css" href="css/METERIOT.icons.css" rel="stylesheet" />

Please notice that the order of the code cannot be changed, because we need JQuery first.

We use .row as one line, also use .col as one column

To give a specific width, use the class .col-TYPE-WIDTH

WIDTH is the integers between 1 to 12, and use 12 means to give a full size(100%)

TYPE is from uss,mp,sscomp,comp,bscomp

We suggest you to add a container outside the row to autosize the inner width

Example:
        <div class="container">
        	<div class="row">
            	<div class="col col-mp-12 col-comp-6 theme-color-blue-primary" style="height:40px;">
                </div>
            </div>
        </div>

What you will see from the example:

If you are using computer, you will find a bar that has only half of the screen's width, but if you are using mobile phone or tablet, you will see the bar is full width.


Cover - Unique Whole-Screen Display Board

Using Cover, you can give users a compatable feeling and a huge attraction, cover is usually in the main page, and we support nesting in the cover

Notice:METERIOT's COVER has default to let its inner elements to be center-aligned, but to let it be free to users, you need to use .text-all-center when you are nesting

Also, if you want to use .row directly in .cover, we suggest you to put a .inner outside the .row

The basic code structure:

<div class="cover">
	<div class="inner inner-top">
		<!-- Top Texts -->
	</div>
	<div class="inner">
		<!-- centered texts -->
	</div>
	<div class="inner inner-bottom">
		<!-- bottom texts -->
	</div>
</div>

Example:

<div class="cover">
	<div class="inner">
		<div class="row">
    		<div class="col col-mp-10">
            	<h1 class="text-all-center">Cover Example</h1>
            	<h2 class="text-all-center">This is the first roar from myself to the world</h2>
        	</div>
    	</div>
	</div>
</div>

METERIOT Navbar

METERIOT has set up a autosize navbar for you

Navbar is made up of a navbar tag

You can put:navicon tag, div.row.navbar-row element inside.

In the .row.navbar-row element, you need to put a .container element to set the width up.

In the element .container, there is .col element, what is to be noticed is that,the .col element does not support the choosing of width as normal, but a special modificatory class .col-navbar

.col element's another modificatory class is .col-navbrand, which means is the title of the site

This is the basic structure for navbar

<navbar class="[Optional Theme Type] [navbar-fixed-top]">
    <navicon></navicon>
    <div class="row navbar-row">
        <div class="container">
            <div class="col col-navbar col-navbrand">
            	[Brand Name]
            </div>
            <div class="col col-navbar">
            	<a href="#">Link1</a>
            </div>
            <div class="col col-navbar">
            	<a href="#">Link2</a>
            </div>
            <div class="col col-navbar">
            	<a href="#">Link3</a>
            </div>
            ....[Can insert more]
            <div class="col col-navbar col-navbar-right"> 
            	<!-- This col will be put on the right -->
            	<a href="#">LinkRight</a>
            </div>
        </div>
    </div>
</navbar>
