How to use it?

.meteriot-animate is your base class, and will needed to be declared on every object that you would like to use METERIOT ANIMATION

.meteriot-animate-hover-zoomout subclass provides a function that will zoom-out slowly when you point your cursor on it.

.meteriot-animate-hover-zoomin subclass provides a function that will zoom-in slowly when you point your cursor on it.

.meteriot-animate-breathe subclass provides a breathing view

.meteriot-animate-jump subclass provides a heartbeat view

.meteriot-animate-rotating-speed subclass provides a rotating effect, the speed is divided into three types, [slow], [normal] and [fast].
